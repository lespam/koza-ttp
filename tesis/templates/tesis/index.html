Programación Genética

Bienvenido a mi aplicación, que busca resolver el cómo organizar una temporada de béisbol de forma óptima, usando una técnica de inteligencia artificial inspirada en la evolución natural.

La Programación Genética consiste en crear y modificar soluciones a partir de una población inicial de programas, usando operaciones como la selección, el cruzamiento y la mutación. Cada programa creado es un árbol de expresión que representa una solución potencial al problema.

La aplicación busca evaluar cada árbol según un criterio de calidad para generar nuevos árboles más prometedores. Así, se puede iterar este proceso hasta encontrar una solución satisfactoria.

Veamos cómo funciona el proceso de selección natural, cruzamiento y mutación cuando se aplican en la búsqueda de soluciones eficientes para el problema de generar un calendario de temporada de béisbol.🧬🤖🔍

Instrucciones

Para comenzar, crea un equipo, una liga, y un torneo. El número de equipos debe ser par.

Después, para generar un calendario de juegos, selecciona un torneo y haz clic en "Generar calendario".
